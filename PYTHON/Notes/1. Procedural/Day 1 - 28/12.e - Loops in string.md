# Loop through a String
Strings are arrays and arrays are [iterable](https://teamtreehouse.com/community/hey-im-confused-on-what-a-iterable-is-what-does-it-mean). Thus we can loop through strings.
	## Example:
```python
alphabets = "ABCDE"
for i in alphabets:
    print(i)

# Output :
A
B
C
D
E
```
